# Smart Group Travel Planner

A modern web application for planning group trips, managing itineraries, and visualizing routes.

## Features

- **Dashboard**: Manage all your trips in one place.
- **Trip Planner**: Day-by-day itinerary management.
- **Interactive Map**: Visualize your destination and activities.
- **Budget Tracker**: Track expenses and visualize spending.
- **Place Discovery**: Search and add places to your itinerary.

## Tech Stack

- **Next.js 14**: App Router
- **TypeScript**: Type safety
- **Tailwind CSS**: Styling
- **Shadcn UI**: Component library
- **Zustand**: State management
- **React Leaflet**: Maps

## Accessing the Application

### 1. Full Application (Next.js)
To use the full travel planner application:

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

### 2. Landing Page Demo
To view the standalone marketing landing page:

- **Option A**: If the dev server is running, visit [http://localhost:3000/landing-demo/index.html](http://localhost:3000/landing-demo/index.html)
- **Option B**: Open the file directly in your browser:
  `public/landing-demo/index.html`
